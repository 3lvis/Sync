## Usage

```objc
if ([NSObject isUnitTesting]) {
    // doing it right!
} 
```

## Installation

**NSObject-HYPTesting** is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'NSObject-HYPTesting'
```
